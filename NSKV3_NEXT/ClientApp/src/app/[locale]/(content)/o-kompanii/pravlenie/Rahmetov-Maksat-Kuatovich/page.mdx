    # Рахметов Максат Куатович
    ### Главный бухгалтер, член Правления

    Образование:
    - 2001-2004 «Академия банковского дела», Экономика, бухгалтерский учет и аудит;
    - 2004-2008 Казахстанский университет «Алатау», Финансы;

    Трудовая деятельность:
    - С 2005 по 2011 работал специалистом по учету, бухгалтером-кассиром, главным бухгалтером филиала АО «Нефтяная страховая компания».
    - С 2011 по 2015 годы переход на государственную службу.
    - С 2015 по 2023 работал главным специалистом, начальником отдела страхового учета, заместителем главного бухгалтера АО «Нефтяная страховая компания».
    - С января 2023 назначен на должность главного бухгалтера, члена правления АО «Нефтяная страховая компания»
